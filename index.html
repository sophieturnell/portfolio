PROJECTS

techTalk

MERN Stack App

1 week, group coding with 3 colleagues

Tech Talk is an app for arranging for tech related meetups. User can create events, see who is attending events and comment on events they are attending. We built the whole app in JavaScript. Our back-end used an Express API to serve the data from a Mongo database and is consumed by our front end built in React.

We used a combination of group coding using Visual Studio Live Share and pair programming to set up the framework, then we split to work on individual elements. Git and Trello were instrumental here for version control and ensuring our code wasn’t duplicated or conflicting. I focused on the forms - styling these with Skeleton, the language buttons - using Adobe Illustrator and Photoshop, and the map element using Mapbox API and react-map-gl.

Scran & Screen

Reacathon (React)

48 hours, coding in pairs

Scran and Screen allows users to search for restaurant recommendations near a chosen location, then shows movie times for nearby cinemas. The app is built using React and uses 2 public RESTful API’s (Zomato and Cinelist) to retrieve the data. We coded collaboratively throughout the entirety of the project sharing a laptop and switching the driver and navigator every 30 minutes. This was the first project where I accessed an API and used Insomnia.

COURSE

Software Engineering Immersive
General Assembly, London

In Autumn 2019, I enrolled in the Software Engineering Immersive course at General Assembly in London.
This was a full-time career accelerator consisting of four modules:


  1. - Programming Fundamentals and Computer Science
  2. - React
  3. - Node & Express
  4. - Python & Django

The intensive course was taught through expert-led lectures, discussions, interactive labs and homework assignments across a variety of languages, frameworks and libraries. Progress was showcased through daily stand-ups, comprehensive portfolio projects and presentations with an emphasis on honing communication and collaboration skills, coding both in pairs and groups.
